# What is this
AdBlock 쓰면 OP.GG에서 열받는 모달창 나오는거 막아주는 프로그램

# How to use
1. 레파지토리(폴더)를 다운 받고 압축해제 합니다.
2. Chrome 브라우저를 열고, 주소창에 chrome://extensions/ 입력 후 엔터.
3. 오른쪽 상단의 "개발자 모드" 토글을 켭니다.
4. "압축 해제된 확장 프로그램 로드" 버튼을 클릭하고, 다운받은 폴더를 선택합니다.
